# CodeShareCopyCat
This is a very simple copy of code share, made possible with socket io and Ace Editor.

It allow the creations of rooms in which the text is shared. 


## Example
The text is only shared between href

![Website](https://i.imgur.com/gyOoXCz.jpg)
## Installing
Clone the repository in a folder of your choice, open a cmd on that folder and type the following commands.

Don't forget to set your certificate on the Certs folder!


    npm install
    node share.js
